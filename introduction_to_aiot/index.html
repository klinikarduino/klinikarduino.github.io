**Introduction to AIoT** **id.handsonembedded.com** **Author:** Erwin Setiawan **[Home Page](../index.html)** Pendaluluan =========================== Tutorial ini bersisi pendahuluan mengenai Internet of Things (IoT) yang digabungkan dengan Artificial Intelligence (AI) atau dikenal juga dengan istilah Artificial Intelligence of Things (AIoT). Tutorial berisi tentang praktik dalam mengimplementasikan sistem AIoT sederhana namun mencakup semua aspek teknologi yang diperlukan untuk membangun suatu sistem AIoT. Beberapa teknologi yang digunakan diantaranya yaitu: * Sensor: DHT11 * Microcontroller: Arduino ESP32 * Data communication: Wi-Fi * Web programming (back-end): embedded web server * Web design (front-end): HTML, CSS (Bootstrap), JavaScript * AI Algorithm: simple linear regression Tujuan =========================== * Pembaca dapat memahami dan menggabungkan semua aspek teknologi yang digunakan untuk membuat suatu sistem AIoT. * Pembaca dapat menggunakan (reuse) kode program dari tutorial ini sebagai template untuk projectnya masing-masing.
